# Create conectado ao Banco de Dados

O projeto é uma Ferramenta de Inserção de Automóveis conectada a um Bando de Dados MySQL em que é possível realizar as seguintes operações:

- Inserir Veículo Doméstico;
- Inserir Moto;
- Inserir Caminhão;
- Inserir Bicicleta
- Inserir Skate;
- Sair do sistema.
